# Sistema de Controle de Estoque de Doces

## Descrição do Projeto

O Sistema de Controle de Estoque de Doces foi desenvolvido para atender às necessidades da **Doceria Gourmet IANES**, que necessita controlar seus produtos de forma digital, substituindo o gerenciamento manual realizado em cadernos.

O sistema permite que usuários autenticados realizem o gerenciamento do estoque de doces, bolos e tortas, possibilitando o cadastro, consulta, atualização e remoção de produtos, além da visualização de itens próximos ao vencimento.

Este projeto foi desenvolvido como um **MVP (Produto Mínimo Viável)**, priorizando simplicidade, rapidez e segurança.

---

## Objetivos

- Controlar o estoque de doces em tempo real.
- Permitir acesso apenas para usuários autenticados.
- Facilitar o cadastro e atualização dos produtos.
- Alertar sobre produtos próximos ao vencimento.
- Registrar informações necessárias para o gerenciamento do estoque.

---

## Funcionalidades

### Autenticação

- Login de usuários.
- Controle de acesso ao sistema.
- Logout do sistema.

### Gestão de Produtos

- Cadastro de doces.
- Listagem de produtos cadastrados.
- Edição de produtos.
- Exclusão de produtos.
- Pesquisa de produtos.
- Ordenação alfabética crescente e decrescente.

### Controle de Validade

- Visualização de produtos próximos ao vencimento.
- Atualização do estoque em tempo real.
